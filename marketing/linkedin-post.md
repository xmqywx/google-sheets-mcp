# LinkedIn Post: Google Sheets MCP Server Announcement

---

I just published something I couldn't find anywhere else — so I built it myself.

**Introducing google-sheets-mcp**: a Model Context Protocol server that gives AI agents full control over Google Sheets through natural language.

There is no official Google Sheets MCP. The closest options are clunky workarounds. So I built one with 14 tools covering everything you actually need:

- Read and write cell ranges
- Create, update, and delete sheets
- Manage spreadsheet structure and formatting
- Query data like a lightweight database

The use case is real: AI agents (Claude, GPT, custom LLMs) can now read reports, populate trackers, update dashboards, and process data in Google Sheets — without a single line of custom integration code.

You describe what you want in plain language. The agent handles the rest.

If you're building AI workflows, automations, or internal tools that touch spreadsheets, this removes a major friction point.

The package is live on npm: **google-sheets-mcp**

Source is on GitHub if you want to dig into the implementation or contribute.
